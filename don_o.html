Windows CE - Don Opperthauser

Abstract

The Chief Technology Officer of Geneer, a leading software development services firm in the Chicago area, will explain Windows CE, Microsoft's small device and embedded systems operating system. Attendees will learn about Geneer's hands-on experiences developing software products and systems with Microsoft's popular new technology.

The presentation will cover several topics:

  • Why CE is different from other embedded operating systems
  • Kinds of devices that run Windows CE
  • Operating system design and application development
  • Programming for Windows CE

Biography

Don Opperthauser is Vice President and Chief Technology Office at Geneer, a leading software development services company that develops software for the Industrial Automation, Financial Services, and Healthcare industries. Don's career as a software professional began as a developer for private education institutions. Over the last 30 years, Don is happy to report he has done nothing but software. Don is a frequent speaker in the software industry and is the author of articles published in DBMS, Control Engineering, and various software development journals.
